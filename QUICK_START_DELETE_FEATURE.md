# 快速啟動 - 刪除關鍵詞回覆功能

## 🚀 立即開始

### 1. 啟動機器人

```bash
python bot.py
```

### 2. 等待同步完成

查看日誌，確認看到：
```
✅ 已全域同步 X 個應用程式命令
✅ KeywordCog 已載入
```

### 3. 測試功能

#### 步驟 1：添加測試關鍵詞
在 Discord 中輸入：
```
/keyword add
```
- 關鍵詞：`測試`
- 回覆內容：`這是測試回覆！`

#### 步驟 2：觸發關鍵詞
在頻道中發送：
```
測試
```

機器人應該回覆：`這是測試回覆！`

#### 步驟 3：刪除回覆
1. 右鍵點擊機器人的回覆訊息
2. 選擇「刪除此回覆」
3. 點擊「確認刪除」按鈕

✅ 訊息應該被刪除！

#### 步驟 4：驗證關鍵詞仍然有效
再次發送：
```
測試
```

✅ 機器人應該再次回覆！

## 🎯 功能說明

### 誰可以刪除？
- ✅ 觸發關鍵詞的用戶
- ✅ 伺服器管理員

### 刪除什麼？
- ✅ 只刪除機器人的回覆訊息
- ❌ 不刪除關鍵詞配置

### 如何使用？
1. 右鍵點擊機器人的回覆
2. 選擇「刪除此回覆」
3. 確認刪除

## ⚠️ 注意事項

1. **Context Menu 同步時間**
   - 全局同步可能需要 1 小時
   - 如果看不到選項，請等待或重啟機器人

2. **機器人權限**
   - 需要「管理訊息」權限
   - 檢查機器人角色設定

3. **只能刪除新的回覆**
   - 舊的回覆（更新前）無法使用此功能
   - 需要重新觸發關鍵詞

## 🐛 問題排查

### 看不到「刪除此回覆」選項？
1. 重啟機器人
2. 等待 Discord 同步（最多 1 小時）
3. 檢查日誌確認 Context Menu 已註冊

### 點擊後沒反應？
1. 檢查機器人是否有「管理訊息」權限
2. 查看日誌中的錯誤訊息
3. 確認是否為機器人的回覆訊息

### 顯示「無權限」錯誤？
- 只有觸發者或管理員可以刪除
- 確認你是否為觸發該關鍵詞的用戶
